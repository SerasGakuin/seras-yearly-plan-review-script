# 年間計画レビューテンプレート

各生徒の年間を通じた記録をするドキュメント、「年間計画レビュー」のスクリプトを定義するGASライブラリ。
個別の生徒用のドキュメントは、このテンプレートを参照する。

### ファイル構成

- 00_番台

関数の実行用のインターフェース関連の実装。

- 01_番台

このライブラリ全体で共有するユーティリティ。

‐ その他の番号

番号ごとに機能が分かれており、互いに参照しあうことはありません。

### 最近の進捗（2026-03-21）

つい最近したことは、

1. 個別の生徒にそれぞれ実装されていたスクリプトをライブラリを参照することで共通化し始めた
2. 共通化にともなって番号わけを含むファイル名の整備
3. メモの項目のレパートリーを固定ではなくその生徒の学習している教科を動的に取得して決定するようにした
